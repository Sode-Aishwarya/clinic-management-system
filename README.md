# 🏥 Clinic Management System

  A Clinic Management System developed using PHP and HTML to help clinics efficiently manage patients, doctors, appointments, staff, and medical records.
  This system provides an organized way to store and retrieve clinic data while reducing manual paperwork.

### 📌 Features

👨‍⚕️ Doctor Management

Add doctor

Update doctor details

View doctor list

Delete doctor

### 🧑‍🤝‍🧑 Patient Management

Add new patient

Update patient information

View patients

Delete patient

### 📅 Appointment Management

Schedule appointments

Update appointment details

View appointments

Delete appointments

### 🏥 Clinic Management

Add clinic details

Update clinic information

View clinics

Delete clinic

### 📋 Medical Records

Add medical records

Update records

View patient medical history

Delete records

### 👩‍💼 Staff Management

Add staff

Delete staff

### 🧪 Tests Management

Add test records
